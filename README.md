# DrMoriarty AudioFlow

## How do I install these formulae?

`brew install drmoriarty/audioflow/<formula>`

Or `brew tap drmoriarty/audioflow` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "drmoriarty/audioflow"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
